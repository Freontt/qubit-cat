# Основная информация
##  Проблематика
Этот проект направлен на создание открытого ПО для передачи информации между людьми при условии полной анонимности. Мы искренне считаем, что в наше время люди не имеют приватности в условиях тотального контроля со стороны крупных организаций и злоумышленников, что вредит как культуре интернета в целом, так и каждому человеку по отдельности.

Наше приложение направлено на решение этой проблемы, предоставляя абсолютно независимую, децентрализованную и защищённую от взлома систему. В приложении сведено к минимуму использование внешних библиотек, благодаря чему возможность взлома стремится к нулю. Также мы готовим решение с расчётом на новые технологии, такие как квантовые компьютеры и нейросети (ботнеты).

## Цель проекта
Создать решение, которое будет отличаться исключительной безопасностью, а также поддерживаться сообществом для сообщества.
## Задачи проекта
1.  Разобраться в протоколах и устройстве сети.
2.  Создать интерфейс для передачи информации по сети.
3.  Обеспечить будущую кроссплатформенность.
4.  Разработать графическую часть для основного интерфейса.
5.  Реализовать или использовать алгоритмы шифрования.
6.  Добавить возможность создания плагинов.
7.  Разработать внешний софт для более удобной модификации основного продукта
8.  Провести цикл тестирований.
## P.S.
Информация будет уточняться и корректироваться в соответствии с развитием.
